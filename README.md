* ladda upp orienteringskarta<br>
* zooma in en tydlig punkt på bådee orienteringskartan och refrenskartan / flygfotot till högar.<br>
* klicka start kalibration<br>
* klicka först på punkten på orienteringkartan<br>
* klicka sedan på referenskartan. Nu syns en röd punkt på orienteringkartan.<br>
* klicka add control point och gör om detta för minstytterligare två andra tydliga punkter som alla bör vara väl spridda.<br>
* om "show overlay" är ikrryssat så kommer orinteringskartan dyka upp ovanpå den högra referenskartan när tre kalibreringspunter är gjorda.<br>
* om felet RMSE är midre än 10 meter så borde orinteringskartan vara väl inpassad.<br>
<br>
* man kan lägga på ett .gpx spår om man vill se hur man sprungit<br>
* man kan exportera till .kmz för att kunna titta på kartan i Googgle Earth eller ladda upp till Iphone appen Orienteering Companion för att lägga en egen bana för träning.<br><br>
