* ladda upp orienteringskarta
* zooma in en tydlig punkt på bådee orienteringskartan och refrenskartan / flygfotot till högar.
* klicka start kalibration
* klicka först på punkten på orienteringkartan
* klicka sedan på referenskartan. Nu syns en röd punkt på orienteringkartan.
* klicka add control point och gör om detta för minstytterligare två andra tydliga punkter som alla bör vara väl spridda.
* om "show overlay" är ikrryssat så kommer orinteringskartan dyka upp ovanpå den högra referenskartan när tre kalibreringspunter är gjorda.
* om felet RMSE är midre än 10 meter så borde orinteringskartan vara väl inpassad.

* man kan lägga på ett .gpx spår om man vill se hur man sprungit
* man kan exportera till .kmz för att kunna titta på kartan i Googgle Earth eller ladda upp till Iphone appen Orienteering Companion för att lägga en egen bana för träning.
